Rules.txt & MOTD.txt Generator

Create formatted server rules and message of the day files with rich text formatting

Formatting Guide:
Sizes: <title>, <large>, <small>
Colors: <red>, <orange>, <yellow>, <green>, <blue>, <purple>, <white>
Close tags: </>
Character Limit: 1024 characters (excluding formatting tags)
Note: Cannot mix size and color tags on the same text. Use multiple color tags per line by closing and reopening.
File Locations:
Place your generated files in:

/home/container/PathOfTitans/Saved/MOTD.txt
